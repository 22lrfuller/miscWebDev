Getting Started with web Developement

There are three programming languages associated with web developement,
HTML, JavaScript, and CSS. HTML changes the way the basics of the website
works, like the text and other simple things. CSS has more to do with
style and images. JavaScript focuses upon functionality rather than looks.
You can write functions and do math in JavaScript.
